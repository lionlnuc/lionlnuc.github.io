经典 Hello World

开始学一门语言的第一步当然是输出 Hello World。

注释

在 Java 中注释的添加和 C++中相同。

记一下快捷键
单行注释：ctrl + /
多行注释：ctrl + shift + /

变量

变量必须先定义，变量名不能重复。
数据类型：

long 8 1234567891011L
float 4 1.2F
double 8 1.2, 1.2 D
boolean 1 true, false
char 2 ‘A’

输入、输出

输入规模比较小时用 Scanner。规模较大时用 BufferedReader。

类型转换

不同类型的数据进行计算，默认采用隐式转换，Java 自动转换，不需要写代码。
转换步骤：

  1. 如有 byte short 类型的数据，先提升为 int 类型。
  2. 把取值范围小的转换为取值范围大的，再进行计算。
    强制转换： 可能会导致精度丢失。
